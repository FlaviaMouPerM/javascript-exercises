# Exercise 16 - Find the Oldest

Given an array of objects representing people with a birth and death year, return the oldest person.

Now that you've reached the final exercise, you should be fairly comfortable getting the information you need from test case(s). Take a look at how the array of objects is constructed in this exercise's test cases to help you write your function.

## Hints
- You should return the whole person object, but the tests mostly just check to make sure the name is correct.
- There are many ways of doing this using built-in array methods like `reduce`, or even chaining multiple! 
- One of the tests checks for people with no death-date.. use JavaScript's Date function to get their age as of today.

# Exercício 16 - Encontre o Mais Velho

Dado um array de objetos representando pessoas com ano de nascimento e falecimento, retorne a pessoa mais velha.

Agora que você chegou ao exercício final, deve estar bastante confortável em obter as informações necessárias a partir dos casos de teste. Observe como o array de objetos é construído nos casos de teste deste exercício para ajudá-lo a escrever sua função.

## Dicas
- Você deve retornar o objeto pessoa completo, mas os testes verificam principalmente se o nome está correto.

- Existem muitas maneiras de fazer isso usando métodos de array integrados, como `reduce`, ou até mesmo encadeando vários!

- Um dos testes verifica pessoas sem data de falecimento... use a função `Date` do JavaScript para obter a idade delas hoje.
