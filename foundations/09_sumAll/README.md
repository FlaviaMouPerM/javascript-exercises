# Exercise 09 - sumAll

Implement a function that takes 2 positive integers and returns the sum of every integer between (and including) them:

```javascript
sumAll(1, 4) // returns the sum of 1 + 2 + 3 + 4 which is 10
```


## Hints

- How will you ensure you're summing all integers within the correct range, no matter the order of the inputs?

- Think about your sum's starting value. Then, how can you make sure every single number from the smaller input to the larger one (including both) gets added to it?

# Exercício 09 - sumAll

Implemente uma função que receba dois números inteiros positivos e retorne a soma de todos os números inteiros entre eles (inclusive):

```javascript
sumAll(1, 4) // retorna a soma de 1 + 2 + 3 + 4, que é 10
```

## Dicas

- Como você garantirá que está somando todos os números inteiros dentro do intervalo correto, independentemente da ordem dos números de entrada?

- Pense no valor inicial da sua soma. Então, como você pode garantir que todos os números, do menor ao maior (incluindo ambos), sejam adicionados a ele?
